# 每天面试题学习

看到各种大佬都在搞每日的面试题，他们只是出面试题，让大家在 issue 里回答。我们就每个工作日整理学习一道面试题，目标还是希望能在这些整理中，学习到更多的知识，提升自己的能力。

- 每天 20 分钟，选择一道题，整理、学习、分享。
- 贵在坚持，坚持就是胜利
- 提升自己，提升能力
